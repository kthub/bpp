# bpp

## Usage

```plaintext
Calculates Bits Per Pixel (BPP) for images

Usage: bpp [OPTIONS] [TARGET_DIR]

Arguments:
  [TARGET_DIR]  Target directory [default: .]

Options:
  -r, --recursive              Recursive search
  -t, --threshold <THRESHOLD>  BPP Threshold (only show files with BPP > threshold)
  -h, --help                   Print help
  -V, --version                Print version
```
